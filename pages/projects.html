How To Add A New Post

You only need to edit data/posts.js. Add one object with a title, type, tags, intro, and sections. 

{
  slug: "my-new-post",
  title: "My New Post",
  type: "Blog",
  date: "2026-03-24",
  readTime: "5 min read",
  excerpt: "Short summary for the card.",
  tags: ["Tag1", "Tag2"],
  intro: "Opening summary for the full post page.",
  sections: [
    {
      heading: "Section Title",
      paragraphs: ["Paragraph one.", "Paragraph two."]
    }
  ]
}
